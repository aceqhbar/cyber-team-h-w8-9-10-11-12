# OWASP Juice Shop – Labs (Structured Report)

---

## Lab 1: SQL Injection — Hidden Data

**Цель**
Получить доступ к скрытым (unreleased) товарам.

**Уязвимость**
Параметр `category` напрямую вставляется в SQL-запрос без фильтрации:

```sql
SELECT * FROM products WHERE category = 'Gifts' AND released = 1
```

**Payload**

```sql
' OR 1=1--
```

**Объяснение**

* `'` закрывает строку
* `OR 1=1` делает условие всегда истинным
* `--` отключает `AND released = 1`

**Результат**
Отображаются все товары, включая скрытые.
